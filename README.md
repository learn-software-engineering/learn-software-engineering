# Learn-Software.com

Website about Software Engineering and Programming.

## Commands

Install build tools:
```
npm install
```

Upgrade all modules:
```
hugo mod get -u ./...
hugo mod tidy
```

## Tools

- [Go](https://go.dev/)
- [Hugo](https://gohugo.io/)
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/)
- [Docsy theme](https://www.docsy.dev/).
- [Font Awesome](https://fontawesome.com/icons).
